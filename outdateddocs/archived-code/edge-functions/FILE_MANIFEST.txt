GETTING STARTED
===============

1. Review Documentation
   - Start with README.md
   - Review DEPLOYMENT_GUIDE.md
   - Check API_EXAMPLES.md

2. Setup Environment
   - npm install -g supabase
   - supabase login
   - supabase link --project-ref <project-ref>

3. Database Migration
   - Run SQL from DEPLOYMENT_GUIDE.md
   - Create indexes
   - Verify table structure

4. WordPress Setup
   - Install plugins (ACF, REST API Menus)
   - Register ACF blocks
   - Create app passwords

5. Local Testing
   - supabase start
   - supabase functions serve
   - Test endpoints

6. Deploy to Production
   - supabase functions deploy
   - Verify in Supabase dashboard
   - Check logs

7. Monitor
   - Check function logs
   - Query database for job status
   - Monitor error details


SUPPORT & MAINTENANCE
====================

Issues? Check:
1. README.md troubleshooting section
2. DEPLOYMENT_GUIDE.md for setup issues
3. Function logs: supabase functions logs <name> --tail
4. Database error_details in siteforge_jobs table

Common Issues:
- WordPress connection: Verify /wp-json/ endpoint
- Media upload: Check CDN URL accessibility
- Block registration: Verify ACF plugin is installed
- Deployment timeout: Check page count and asset size
