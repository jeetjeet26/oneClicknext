INSTALLATION INSTRUCTIONS
--------------------------

1. Upload oneclick-siteforge/ to /wp-content/themes/
2. Activate theme in WordPress Admin
3. Install ACF Pro if not already installed
4. Go to Dashboard > Theme Settings
5. Fill in required configuration
6. Create your first page and add blocks
7. See QUICKSTART.md for detailed instructions
